# 전국 고등학교 지도 플랫폼 기획서

## 1. 프로젝트 개요

### 프로젝트명 (가칭)
HiMap – 전국 고등학교 지도 서비스

### 목적
- 전국의 모든 고등학교를 카카오 지도를 통해 **한눈에 시각화**


### 타겟 사용자
- 교육솔루션 회사

---

## 2. 핵심 컨셉

- **지도 중심 UI**
- 검색보다 “탐색”에 강한 서비스
- 학교는 점(Point)이 아니라 **정보 단위(Card)** 로 취급
- 필터 → 비교 → 상세 정보 흐름

---

## 4. 데이터 구성

### 고등학교 기본 정보
- 학교명
- 학교 유형 (일반고 / 특목고 / 자사고 / 특성화고)
- 설립 구분 (공립 / 사립)
- 주소
- 위도 / 경도
- 남녀공학 여부
- 홈페이지 URL

---

## 5. 주요 기능 정의

## 5.1 지도 뷰 (메인)

### 기능
- 전국 고등학교 마커 표시
- 줌 레벨에 따른 마커 클러스터링
- 지역 이동 시 자동 데이터 갱신

### UX
- 마커 클릭 → 학교 미리보기 카드

---

## 5.2 학교 검색

### 검색 기준
- 학교명
- 지역 (시/도, 구)
- 학교 유형
- 설립 구분

### UX
- 자동완성 검색
- 검색 결과 지도 중심 이동

---

## 5.3 필터 기능

- 학교 유형 필터
- 남/여/공학
- 공립 / 사립
- 지역 필터

> 필터 변경 시 지도 즉시 반영 (Client-side filtering)

---

## 5.4 학교 상세 페이지

### 구성
- 학교 기본 정보
- 지도 위치
- 학교 유형 태그
- 공식 홈페이지 링크
- 비교 대상 추가 버튼

---

## 6. 카카오 지도 연동 전략


- CSR 전용 컴포넌트 (`dynamic import, ssr:false`)
- 마커는 Canvas 기반 렌더링
- 클러스터는 Kakao MarkerClusterer 사용
- 좌표는 미리 정규화된 데이터 사용

---

## 7. 성능 고려 사항

- 지도 이동 시 Debounce 처리
- 줌 레벨별 데이터 제한
- 학교 데이터는 지역 단위 캐싱
- 마커 렌더링 최소화
